# API

| Property  | Description | Type          | Default |
| --------- | ----------- | ------------- | ------- |
| children  |             | ReactNode     | -       |
| className |             | string        | -       |
| style     |             | CSSProperties | -       |
| title     |             | ReactNode     | -       |
| extra     |             | ReactNode     | -       |
| showArrow |             | boolean       | `true`  |

## Demo

```javascript
<Accordion title="Accordion Title">
    <div>Accordion Content</div>
</Accordion>
```
